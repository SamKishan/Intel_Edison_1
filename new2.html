This project is part of the Secure Embedded Programming (TLEN 5830) course taught at CU boulder. The web server is being run on Intel Edison and is making use of the Flask service.

  • The "Display the temperature(F)" option reads the temperature from a temperature sensor installed on the Edison and displays it.
  • The "Button Status" option displays the status of the switch. It could either be "pressed" i.e closed or "not pressed" i.e open.
  • The "Control 1st LED" option lets a remote user control an LED; switch between ON and OFF.
  • The "Display LED status" option displays the status of the LED on board
  • The "Control 2nd LED" option lets a remote user control the 2nd LED; switch between ON and OFF
